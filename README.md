# Typer _ Cli Development Guide
step by step guide for building , styling, and publishing Python CLI tools using Typer &amp; Rich

## 1-installation 
1-create a folder called typer and a file calld cli_1 within it create your first hello_world.py
```bash
typer/
 └── cli_1/
       └── hello_world.py
```
2-in Vs code terminal , run this to create virtual environment (powerShell)
```bash
.\.venv\Scripts\activate
```
3- now install typer
```bash
pip install typer
```
3-After installation, you should see
```
(.venv)
```
